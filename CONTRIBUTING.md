# Contributing

Thanks for your interest in Forge Language Support.

## Open Access

This repository is free and open to everyone.

If you want to correct, modify, or improve any part of the codebase, your contribution is welcome and genuinely appreciated.

## How You Can Help

- Fix bugs
- Improve code quality and structure
- Improve tutorials and documentation
- Add tests and examples
- Suggest better patterns for language tooling

## Contribution Flow

1. Fork the repository
2. Create a feature/fix branch
3. Make your changes
4. Open a Pull Request with a clear description

If you find a problem but cannot submit code, opening an issue is still very helpful.

Thank you for helping make this project better.
